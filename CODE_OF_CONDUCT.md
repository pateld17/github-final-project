# Code of Conduct

## Our Pledge
We pledge to create a welcoming and respectful environment for everyone.

## Our Standards
Examples of positive behavior include respectful communication and constructive feedback.

## Enforcement Responsibilities
Maintainers are responsible for enforcing the standards.

## Scope
This Code of Conduct applies to all project spaces and public interactions.

## Enforcement
Instances of abusive behavior may be reported and reviewed.

## Enforcement Guidelines
Violations may result in:
- Warning
- Temporary ban
- Permanent ban depending on severity

## Attribution
This Code of Conduct is adapted from standard open-source guidelines.
